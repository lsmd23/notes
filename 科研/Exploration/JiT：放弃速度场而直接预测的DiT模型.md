# 文献内容
## 一、 必要的前置工作
- 扩散模型最初的开发核心思想是“去噪”，即从损坏的图像版本中预测出干净的图像。
- 随后扩散模型的发展出现了两个重要里程碑，导致其偏离了直接预测干净图像的目标：一是DDPM确立了预测噪声本身（$\epsilon$-prediction）的范式，二是流匹配（Flow Matching）等方法引入了预测结合了数据与噪声的速度变量（$v$-prediction）。
- 为了在像素空间中进行高维扩散建模并避免信息瓶颈，现有的模型大多依赖密集卷积（如U-Net的过完备通道和长跳跃连接）或具有层次结构的小patch设计，这些方案往往缺乏通用Transformer的极简性。
## 二、 其解决的问题
- **核心痛点（流形假设）**：根据流形假设，自然的高维数据实际上位于低维流形上，而噪声$\epsilon$或速度$v$本质上是偏离流形的，分布在整个高维空间中。
- **维度灾难**：在面对高维观测空间（如原始像素）时，要求模型预测高维的噪声或速度需要极高的网络容量以保留所有的噪声信息；这导致标准的Vision Transformer在处理高维输入时会面临灾难性的性能崩溃。

## 三、 关键数学推导
文献采用基于流匹配和ODE视角的公式，核心数学推导如下：
- **变量与前向过程**：给定真实数据$x \sim p_{data}(x)$和噪声$\epsilon \sim p_{noise}(\epsilon)$，在训练期间，时间$t \in [0, 1]$时的带噪样本$z_t$被定义为线性插值：$$z_t = t x + (1-t)\epsilon$$
- **流速度（Flow Velocity）定义**：流速度$v$被定义为$z_t$对时间的导数（即$v_t = z_t'$），结合上式可得：$$v = x - \epsilon$$
- **预测空间转换**：如果让网络的直接输出为干净数据$x_\theta$（即$x_\theta = net_\theta(z_t, t)$），基于上述两个方程，我们可以解出网络隐式预测的噪声$\epsilon_\theta$和速度$v_\theta$：$$\epsilon_\theta = (z_t - t x_\theta) / (1-t)$$$$v_\theta = (x_\theta - z_t) / (1-t)$$
- **训练目标（$v$-loss空间）**：该算法最终采用$x$-prediction配合$v$-loss进行优化，这种组合本质上是对$x$-loss的一种重新加权：$$\mathcal{L} = \mathbb{E}_{t, x, \epsilon} ||v_\theta(z_t, t) - v||^2$$其中代入预测公式即为$v_\theta(z_t, t) = (net_\theta(z_t, t) - z_t) / (1-t)$。
## 四、 其对应的新架构
本文提出了一种完全自包含的“仅图像Transformer”（Just image Transformers，简称JiT）架构，该架构本质上是直接将普通的ViT应用于原始像素Patch上。
![[Pasted image 20260406192445.png|395]]
- **组件实际目的/意义**：
    - **Raw Pixel Patches**：直接使用极大尺寸的原始像素块（如$16\times16$或$32\times32$），彻底抛弃Tokenizer、预训练对齐和感知损失等外部依赖。
    - **Linear Embed (Bottleneck Layers)**：摒弃常规的全维线性映射，采用两层包含低维瓶颈（如降至128维再升维）的线性层；其意义在于强制模型进行信息压缩，符合流形学习中鼓励有用信息通过的原则。
    - **$x$-prediction**：要求网络仅保留低维数据信息并滤除噪声，使得容量有限的网络也能在极高维度的空间中有效运作。
## 五、 实验结果与得出的一些启示
- **预测目标是克服高维灾难的关键**：在处理高维Patch（例如$256\times256$分辨率下的768维Patch）时，$\epsilon$-prediction和$v$-prediction无一例外地发生了灾难性失败，而$x$-prediction在各种损失函数空间下均能表现出色。
- **无限增加网络宽度并非必要**：实验证明，即使Patch维度（如高达3072维或12288维）远超Transformer的隐藏层维度，$x$-prediction也能正常工作，这意味着网络设计可以很大程度上与观测维度解耦。
- **瓶颈结构具有正向收益**：在网络中引入降维的Bottleneck不仅没有破坏生成能力，反而带来了高达约1.3 FID的质量提升，这与经典流形学习的观察高度一致。
- **卓越的跨分辨率扩展性**：通过简单地按比例放大Patch大小并同等缩放噪声尺度（例如在$512\times512$时将噪声放大2倍），JiT模型可以在保持计算量（FLOPs）和序列长度基本不变的情况下，直接扩展到高分辨率生成任务中。
