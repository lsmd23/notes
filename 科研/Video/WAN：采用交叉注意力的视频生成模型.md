# 文献内容
## 1. 必要的前置工作
- 视频生成技术在闭源模型（如Sora、Kling）的推动下取得了显著进展，但开源模型与闭源商业模型在性能、多任务处理能力及计算效率上仍存在明显差距。
- 当前的视频生成框架普遍基于Stable Diffusion架构演进，或者采用在视觉生成任务中表现优异的扩散Transformer（DiT）结合流匹配（Flow Matching）范式。
- 为了将高维视频映射到紧凑的潜在空间，主流方案广泛采用变分自编码器（VAEs）处理空间与时间维度。
- 高性能的文本到视频（T2V）生成极度依赖如T5系列或多模态大语言模型（MLLMs）等强大的文本编码器来实现视觉与语义的精确对齐。
## 2. 其解决的问题
- **弥合开源与商业模型的性能鸿沟**：通过扩展预训练数据规模与模型参数（最高至14B），Wan在动态质量、物理合理性、视觉细节和指令遵循方面全面超越了现有的开源和闭源商业模型。
- **打破单一生成功能的局限性**：Wan不仅提供基础的T2V功能，还通过统一框架支持了图像生成视频（I2V）、指令引导视频编辑（V2V）、零样本个性化定制、相机运动控制及实时流式视频生成等八项核心下游任务。
- **突破多语种视觉文本生成瓶颈**：解决了过往视频模型难以渲染文字的缺陷，Wan是首个能够在动态画面中同时精准生成中文和英文视觉文本的模型。
- **解决高昂的算力门槛问题**：团队在推出14B大模型的同时，专门设计了1.3B参数的高效版本。该版本仅需8.19 GB显存即可在普通消费级GPU上运行，且评测性能依然超越了许多大参数开源模型。
## 3. WAN的全技术栈与新架构设计
以下是WAN模型架构及核心组件意义的建模总结：
![[Pasted image 20260405195619.png]]
- **Wan-VAE系统设计**：采用新颖的3D因果变分自编码器架构，将所有**GroupNorm层替换为RMSNorm**以严格保持时间因果关系。引入了特征缓存机制（Feature Cache），按块（Chunk）处理潜特征，能够有效防止显存溢出并支持无限长视频的高效编解码。
	- 替换归一化，提升缓存结果的利用效果，保持时间的因果性
	- **对第一帧，不做时间压缩，兼容图像数据**
	- 设计不同的卷积方式，用于不同的压缩比例
	- **先在二维图像上进行训练，再在三维上进行训练**，提供空间上的压缩先验，以确保训练不会失真
	- 调优过程借鉴GAN过程进行调优
- **主干网络（DiT）与流匹配**：网络彻底摒弃U-Net，完全基于全时空注意力的Transformer块构建。模型通过流匹配对图像和视频进行联合训练，定义插值潜变量为 $x_t = t x_1 + (1-t) x_0$，核心优化目标是最小化模型输出速度 $u(x_t, c_{txt}, t; \theta)$ 与真实目标速度 $v_t = x_1 - x_0$ 之间的均方误差损失 $\mathcal{L}$。
	- 采用Cross-Attention处理多模态数据，更注重完整的视频模块注意力，避免文本模态数据干扰视频的生成模态
- **工程加速与并行策略**：训练期采用全切片数据并行（FSDP）与二维上下文并行（外层Ring Attention + 内层Ulysses）结合的策略，完美解决超长序列的显存与通信瓶颈。推理期深度集成了扩散缓存（利用相邻步与无条件生成的注意力相似性复用计算）、FP8 GEMM以及优化后的8-bit FlashAttention技术实现极致加速。
	- 并行训练DiT加速训练的策略
- **Caption Model标注策略**：借助LLaVA架构，采用先进的V2T模型，按3fps抽帧（最多129帧），之后进行池化压缩，编码提取关键信息。设计了如下的多种模式数据，确保对视频有完整的效果
	- counting
	- OCR
	- camera angle & motion
	- fine-grained categories
	- relational understanding
	- editing instruction
	- group image description
	- human-annotated dense captions
	- ……
## 4. 实验的结果与启示
- **全面领先的评测表现**：在严苛的Wan-Bench与主流VBench评测中，Wan-14B的综合加权得分（涵盖大运动生成、物理合理性、单/多对象准确度等）一致优于Sora、Kling及一众开源模型。
- **自适应归一化（adaLN）的设计启示**：消融实验揭示，在给定参数预算时，让所有Transformer块“完全共享全局adaLN参数并将冗余参数用于增加网络深度”，比为每个块保留独立adaLN参数更能有效降低训练损失并提升收敛效果。
- **文本编码器架构的选取启示**：在视频生成这种高度依赖语义空间映射的任务中，具有双向注意力机制的**umT5模型**比仅具备因果注意力的纯Decoder大语言模型（如Qwen2.5和GLM-4）表现出更优越的文本特征对齐能力与生成质量。
	- 放弃MLLM的编码器机制，恢复使用传统双向编码器作为文本条件的来源（这类编码器的双向特征更加契合diffusion的嵌入需求）
- **Visual Text Data的生成启示**：Wan通过给模型喂大量的单纯汉字/英文图像，不能并结合OCR抽取文本，嵌入文本编码器中描述，用于模型的预训练，成功解决了过往视频生成模型无法渲染清晰文本的难题，实现了在动态视频中同时生成高质量中文和英文视觉文本的能力。
- **零外部依赖的个性化生成启示**：模型验证了无需依赖ArcFace等外部特征提取器，只需在Wan-VAE潜空间内应用自注意力机制，将用户参考面部图像直接拼接进扩展帧中进行去噪，即可实现无损的高保真视频个性化定制。
- **无限长视频与实时流媒体的商业化启示**：通过将基础DiT改造为滑动时间窗口（Streamer）推理机制，并结合潜在一致性模型（LCM）进行降步蒸馏以及TensorRT量化，Wan成功在单张消费级RTX 4090 GPU上达到了20 FPS的实时生成速率，彻底打通了高质量视频模型在互动娱乐与直播环境中的落地路径。
