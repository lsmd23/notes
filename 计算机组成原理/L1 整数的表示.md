# 位与信息
- 计算机中，用0/1的二进制表示数字和信息，每个0/1叫做**位**
- 每个**字节**，由8个位组成，是表示数的常用的单位，也是多数计算机的**可寻址的最小存储单位**
- 每个字节都对应了唯一的一组编码，表示它在计算机中的位置，称为**地址**
- **字长**：计算机中用于表示地址的位数，常见的有32位和64位，字长同时也意味着**虚拟地址空间的大小**
- 字节的顺序：大端法和小端法
	- 变量一般都不会仅占一个字节，而是多个字节
	- 大端法：高位字节存放在低地址处
	- 小端法：低位字节存放在低地址处
